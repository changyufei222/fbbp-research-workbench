# Example Research Requests

## Request 1: Landscape Review
"请帮我调研 FBBP 领域中 knottin scaffold 的典型特征、常见靶点、来源文献，以及它在展示随机肽或靶向结合中的典型用途。请输出一份带引用的中文研究摘要。"

## Request 2: Candidate Comparison
"比较两个候选 scaffold 在稳定性、结构支持证据、适配靶点范围和亲和力数据覆盖上的差异，并指出目前证据还缺什么。"

## Request 3: Methodology Explainer
"解释数据库中 PDB 结构质量指标在条目筛选中的作用，说明这些指标如何支持或限制后续分析结论。"

## Request 4: Research Planning
"围绕主链柔性靶向结合蛋白数据库，给我制定一个研究计划：先查哪些来源、要回答哪些核心问题、最后报告应该怎么组织。"

## Request 5: Private + Public Evidence Chain
"先使用私有 FBBP 知识库总结目标 scaffold 的关键事实，再补 PubMed 文献和 UniProt/PDB 元数据验证，最后输出一份带 tool trace 的研究备忘录。"
